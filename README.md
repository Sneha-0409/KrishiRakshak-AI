# KrishiRakshak-AI - An AI-Powered Crop & Livestock Disease Diagnosis and Reporting Platform

## Core Features
1. Farmer App
- Register/Login
- Upload crop or livestock image
- Enter symptoms (optional)
- Get disease prediction
- Receive treatment suggestions
- View previous reports

2. AI Engine
- Disease classification
- Confidence score
- Preventive measures
- Treatment recommendations

3. Admin Dashboard
- View disease reports
- Track disease hotspots
- View analytics

## Design Architecture
Flutter App
      ↓
FastAPI Backend
      ↓
AI Model (TensorFlow)
      ↓
PostgreSQL Database
      ↓
Admin Dashboard (React)
